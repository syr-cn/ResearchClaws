📖 DNL 笔记预览 (MemAgent-RL)
Motivation: 现有 agent 缺乏主动记忆管理，记忆操作依赖 prompt engineering
Method: 用 PPO 训练 memory controller — 学习 read/write/forget 三种 action
Key Result: WebArena +18% · ALFWorld +23% over SoTA
Limitation: 固定 memory slot 数量(K=16)，无法适应不同任务复杂度
Relevance: 直接可扩展 → 自适应 K 是明确 gap

📝 AdaptMem 研究提案 (v3)
💡 Core Idea
RL-trained memory controller with dynamic capacity allocation — agent 自主决定 K 值、淘汰策略和记忆粒度
🏗 Method Overview
① 扩展 MemAgent-RL 为可变 K
② reward shaping: task success + memory efficiency
③ curriculum: 简单任务→复杂任务 progressive training
📅 Timeline (4 weeks)
W1: 复现 baseline · W2: 可变 K 实现
W3: RL training + ablation · W4: 写作
⚠️ Risks
R1: 可变 K 导致 RL 不稳定 → 渐进式课程学习
R2: 评测公平性 → 控制总 memory token 预算
