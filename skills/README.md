# ResearchClaw Skills

6 atomic skills covering the full research workflow.

| # | Skill | Trigger Words | Description |
|---|---|---|---|
| 1 | [paper-scout](paper-scout/) | `推荐今日论文` `paper scout` `每日论文` | Daily arXiv paper discovery with personalized ranking |
| 2 | [paper-reader](paper-reader/) | `帮我读一下` `DNL` `论文速读` + arXiv link | Deep reading with Deep Note 7-section framework → markdown note (HTML opt-in) |
| 3 | [reading-list](reading-list/) | `我的论文列表` `reading list` `加入待读` `标记已读` | Kanban-style paper list management + HTML dashboard |
| 4 | [research-profile](research-profile/) | `我的研究画像` `更新研究画像` `research profile` | Research taste profile maintenance + visualization |
| 5 | [idea-generator](idea-generator/) | `给我一些研究灵感` `idea generator` `跨论文分析` | Cross-paper gap analysis → 3-5 research idea proposals |
| 6 | [paper-writer](paper-writer/) | `论文大纲` `写引言` `审稿` `rebuttal` | Full paper writing: outline → draft → review → rebuttal |

## Shared Resources

- `../templates/` — HTML templates for paper notes, reading list, and research profile
- `~/.openclaw/workspace/research-claw-config.md` — User research profile config
- `~/.openclaw/workspace/research-claw-reading-list.json` — Reading list data
- `~/.openclaw/workspace/research-claw-output/` — Generated HTML output directory
